__Oval__ is a browser extension that can summarizes the web page you are browsing. What it differs from other summarizerish extension is that it has an experience that is similar to ["Pinch to Summarize"](https://www.reddit.com/r/ArcBrowser/comments/1ay42ra/pinch_to_summarize_behind_the_scenes/) from [Arc Search](https://arc.net/search). <small>It is also available for Firefox and Edge.</small>

Check it out on <a href="https://chromewebstore.google.com/detail/oval/kogclmjmcggincppnnmdgacjmjmnlhhj">Chrome Web Store</a>
