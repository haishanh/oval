__Oval__ is a browser extension that can summarizes the web page you are browsing. What it differs from other summarizerish extension is that it has an experience that is similar to ["Pinch to Summarize"](https://www.reddit.com/r/ArcBrowser/comments/1ay42ra/pinch_to_summarize_behind_the_scenes/) from [Arc Search](https://arc.net/search).
